# ДЗ-01. Архитектура ПО (C4 / Structurizr DSL)

## Вариант 15 — Управление библиотекой

Система предназначена для учета пользователей, книг и операций выдачи/возврата книг.

### Основные сущности
- Пользователь
- Книга
- Выдача

### Основные функции (API)
1. создание пользователя
2. поиск пользователя
3. добавление книги
4. поиск книг
5. создание выдачи книги пользователю
6. получение списка выданных книг пользователю
7. возврат книги
